# mio-lifepilot – Runder Tisch der KIs

## Setup
1. `npm install`
2. Keys eintragen (OpenAI, GitHub PAT)
3. `npm run debate` – Tisch startet

## Wie es funktioniert
- 7 KIs debattieren Input in <60 Sek.
- Konsensus pusht Code zu GitHub.
- Selbst-Optimierung: Wöchentliche Loops.

Beispiel: `node debate.js` – Output: Konsensus-Code.
